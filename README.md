# 概要

こおりのぬけみち-like な迷路の訓練用 WEB アプリケーション.

# リンク

[https://f-mm7.github.io/ice-path](https://f-mm7.github.io/ice-path/)

# ルール

マスの色と意味は以下の通り
| 色 | 意味 |
| - | - |
| white | 空 |
| black | 岩 |
| lime | 駒の現在地 |
| red | 目的地 |

十字キーで方向を指定すると,
駒はその方向に岩か縁にぶつかるまで移動します.

数字は移動可能回数を示します.

目的地に静止することができればクリアです.

↺ はリセットボタンです.
